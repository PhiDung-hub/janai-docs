# `cortex run`

Starts an interactive chat shell with a model.

## Usage

```bash
cortex run MODEL_ID
```

## Command Chain

`cortex run` is actually syntactic sugar that invokes the following command chain. 

1. `cortex start MODEL_ID`
2. `cortex chat`
