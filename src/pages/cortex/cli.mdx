# Cortex

Cortex is a CLI tool used to interact with the Jan application and its various functions.

<Callout type="info">
Cortex CLI is compatible with all OpenAI and OpenAI-compatible endpoints.
</Callout>

## Usage

```bash
cortex [command] [flag]
```
### Options
```
  -h, --help       display help for command
```
## Sub Commands
-  [models](/cortex/cli/models): Manage models.
-  [serve](/cortex/cli/serve): Start an API endpoint server for Cortex backend
-  [chat](/cortex/cli/chat): Send a chat request to a model
-  [init|setup](/cortex/cli/init): Init settings and download cortex's dependencies
-  [ps](/cortex/cli/ps): Show running models and their status
-  [kill](/cortex/cli/kill): Kill running cortex processes
-  [pull|download](/cortex/cli/pull): Download a model. Working with HuggingFace model id.
-  [run](/cortex/cli/run): Shortcut to start a model and chat **(EXPERIMENTAL)**.
