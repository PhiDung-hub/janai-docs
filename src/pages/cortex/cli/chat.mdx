# `cortex chat`

This command initiates a Chat Completion inference request using a specified model. This command primarily opens an interactive chat shell where you can interact directly with the model.

## Usage

```bash
cortex chat --model MODEL_ID
```
### Options
```
  -m, --model <model_id>  Model Id to start chat with
  -h, --help              display help for command
```
