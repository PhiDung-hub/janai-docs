This article lists the capabilities of the Jan platform and guides you through using RAG to chat with PDF documents, web and desktop searches.

## Retrieval

To chat with PDFs using RAG in Jan, follow these steps:

:::caution
Please turn on Experimental mode in Settings to use this feature. [Read more](/guides/advanced/#enable-the-experimental-mode).
:::

1. Create a **new thread**.
2. Check that **Tools** > **Retrieval** is toggled on.

4. Upload your PDF.
5. Chat with your PDF.
